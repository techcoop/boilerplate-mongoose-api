# boilerplate-mongoose-api
Yet another basic express / mongoose REST API boilerplate with auth0 JWT

# Requirements
* Node >= 8.0.0
* Yarn >= 0.25.0 or NPM >= 6.0.0
* Should work on Linux, OSX, Windows

# Installation

```bash
yarn
```

# Development
```bash
yarn start
```

# Testing
```bash
yarn test
```
